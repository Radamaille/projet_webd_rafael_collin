# Tesseract Library

Ce site Web à été développé en React + TypeScript et permet de rechercher des Livres (précisément ou non) avec l'API OpenLibrary.
Elle permet également de consulter les dernières mise à jour de la librairie du site.

---

#Fonctionnalités 
- Barre de recherche de livres 
- Recherche avancée permettant de rentrer des détails spécifiques sur une oeuvre
- Affichage des changements récents de la librairie
- Navigation entre la Home page et la recherche avancée avec React Routeur
- Tests unitaires sur les fonctionnalités principales

---

# Technologies utilisée
- **React**
- **TypeScript**
- **Vite**
- **React Router**
- **Jest**
- **@testing-library/react**
- **API OpenLibrary**

---

## Etapes de lancement du Projet en local : 

### 1) Installer les dépendances
```bash
npm install
```
### 2) Lancer le serveur web
```bash
npm run dev
```
### 3) Lancer les tests
```bash
npm test
```
